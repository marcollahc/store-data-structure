Tema do trabalho: Gerenciamento para um Acervo de obras de arte
Nosso grupo escolheu criar um Gerenciamento para um Acervo de obras de arte, então são peças únicas, em que cada id representa unicamente uma peça.

As estruturas que nós utilizamos foram:
1- Lista encadeada para o armazenamento dos pedidos

2- Pilha para análise dos pagamentos

3- Fila para expedição dos pedidos 
Lista dos pedidos a serem expedidos, composta por id, valor total e situação.

4- Árvore binária para o estoque dos produtos
Lista dos produtos que serão vendidos pela loja, composta por id, largura, altura, comprimento e local.
